*****************
Other Instruments
*****************

This section of the example gallery showcases how to create and synthesize
other instruments using the line-list and response machinery in ``muse``.
While this machinery lives in ``muse``, it is actually instrument-neutral.
The MUSE-specific information  lives in the detector mapping (``map_response_to_sg_detector``).

All examples require a local CHIANTI database configured with ``XUVTOP``.
